# 05. Practice

## 实战目标

通过一个最小可运行案例，完成从 Agent 到 Harness 的工程闭环。

## 推荐步骤

### Step 1：构建基础 Agent

- 输入用户问题
- 生成结构化响应
- 记录关键日志

### Step 2：加入工具调用

- 增加检索 / API / 数据查询能力
- 对工具调用做失败重试与超时限制

### Step 3：构建 Harness（核心）

- 固定测试样本
- 定义输出校验规则
- 批量执行并输出报告

### Step 4：接入评估系统

- 指标：准确率、一致性、latency、成本
- 形成可对比基线

## 产出物建议

- `examples/`：可运行示例
- `resources/`：样本数据
- `diagrams/`：流程图与架构图
