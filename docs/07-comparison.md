# 07. Comparison

## 与其他范式对比

| 范式 | 优点 | 局限 |
| --- | --- | --- |
| Prompt Engineering | 启动快、门槛低 | 难维护、难评估 |
| Fine-tuning | 任务适配强 | 成本高、迭代慢 |
| Agent Engineering | 编排灵活 | 稳定性不足 |
| Harness Engineering | 可控、可测、可评估 | 建设成本较高 |

## 适用建议

- 个人试验：Prompt Engineering
- 任务增强：Agent Engineering
- 规模化交付：Harness Engineering

## 结论

Harness Engineering 是将 AI 能力产品化、工程化、平台化的关键路径。
